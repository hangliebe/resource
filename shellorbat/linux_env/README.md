# configvim使用指南

上面要给windows上的git Bash或者linux上配置一个比较好的vim操作环境，可以参考以下步骤：

- 右键管理员身份打开`Git Bash`

>使用管理员身份的原因是要往/wingw64//bin下面拷贝文件时需要文件夹操作权限。

- 执行脚本

```
wget https://gitee.com/hangliebe/resource/raw/master/shellorbat/linux_env/configvim.sh
```

```
sh configvim.sh
```
